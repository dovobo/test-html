Settings = CTRL + ,
Duplicate Line  = Ctrl + Shift + D
Zeile verschieben = Ctrl + Up (or Down)
Select the next matching characters = markieren und CTRL + D
Select all matching characters = ALT + F3
Toggle comments (on and off) = Ctrl + /
Zeichenfolge löschen = Ctrl + Backspace
Text vergrößern/ verkleinern = Ctrl + + or -

Für die autocomplet ohne < schreiben
